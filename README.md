# 📝 Smart Notes Manager

## 👤 Author
Ayush Rai

## 📌 Description
A feature-rich notes management web app that allows users to add, edit, delete, and search notes. It also supports dark mode and stores data using browser localStorage.

## 🚀 Features
- Add notes
- Edit notes
- Delete notes
- Search notes
- Dark mode toggle
- Persistent storage using localStorage

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in your browser

## 💡 Future Scope
- Cloud storage integration
- User authentication
